# kopo21_Java

Ticketing Machine

ㄴ  LotteWorldTicketing.java : main 클래스
  
ㄴ  TicketSystem.java : main 클래스로 실행되는, 반복문과 함수를 부르는 클래스
  
ㄴ  AskStuff.java : 사용자에게 여러가지를 물어보는 함수가 담긴 클래스
  
ㄴ  CalculateStuff.java : 받은 입력값으로 출력값을 계산하는 클래스
  
ㄴ  PrintStuff.java : 출력값을 print하고 csv파일에 기록하는 클래스
  
ㄴ  OrderData.java : 출력값을 위한 상수가 모여있는 클래스
  
ㄴ  StaticValue.java : 변하지않는 상수가 모여있는 클래스

 ㄴ ReportStuff.java : 결과값을 print하는 클래스
